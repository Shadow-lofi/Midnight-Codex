House Rules

  1. Build one another up. Critique the code, never the coder. (Eph 4:29)
  2. Keep it Christ-centered. A fellowship first, a dev group second.
  3. Help before you scroll. If you can answer a question, do.
  4. Give credit, keep it clean. No piracy, no plagiarism, no spam.
